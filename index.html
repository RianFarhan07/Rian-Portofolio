My Skills

Kotlin

Bahasa pemrograman modern untuk pengembangan aplikasi Android, yang lebih ringkas dan aman dari Java. Kotlin kompatibel dengan Java dan mendukung fitur-fitur fungsional.

Firebase

Platform cloud dari Google yang menyediakan layanan backend seperti database, autentikasi, dan notifikasi untuk aplikasi web dan mobile. Firebase memudahkan pengembangan tanpa perlu mengelola server sendiri.

Java

Bahasa pemrograman berorientasi objek yang digunakan di berbagai platform, termasuk pengembangan Android. Java dikenal stabil dan portabel ("write once, run anywhere").

HTML

Bahasa markup untuk membuat struktur dan konten halaman web. HTML digunakan untuk menyusun elemen seperti teks, gambar, dan link.

CSS

Bahasa yang mengatur tampilan visual halaman web, termasuk layout, warna, dan font. CSS memisahkan konten dari desain untuk memudahkan styling.

Javascript

Bahasa pemrograman yang berjalan di browser untuk membuat halaman web interaktif. JavaScript bisa memanipulasi elemen HTML dan merespon interaksi pengguna.
